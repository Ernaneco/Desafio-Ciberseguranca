# Desafio-Ciberseguranca

Seguindo as orientações propostas pelo desafio, foram realizados os procedimentos no sistema Kali Linux, via máquina virtual, para invasão à interface metasploit, conforme arquivos em anexo.

Os trẽs primeiros arquivos referem-se à tentativa de conexão FTP utilizando a ferramenta medusa para p ataque.

Os arquivos de número 4 a 6 abrangem o ataque efetuado para login no site dvwa, novamente utilizando a ferramenta medusa.

Os demais arquivos contemplam o método de enumeração para acesso ao protocolo SMB.
